# Hermes Repository

A Kodi 19+ repository hosting **Hermes** — a fork of [Twilight](https://github.com/kodi7rd) with
[CocoScrapers](https://github.com/CocoJoeTV/script.module.cocoscrapers) embedded as a built-in
scraper module (`ScrapperHermes`). Single addon, no external dependencies.

## Install

1. Download [`repository.hermes-1.0.1.zip`](https://hermestelemachus.github.io/hermes/repository.hermes-1.0.1.zip)
2. Open Kodi → **Add-ons → Install from zip file** → pick the downloaded zip
3. **Add-ons → Install from repository → Hermes Repository → Video add-ons → Hermes → Install**
4. Updates are then handled automatically by Kodi.

If "Install from zip file" is disabled, enable **Settings → System → Add-ons → Unknown sources** first.
